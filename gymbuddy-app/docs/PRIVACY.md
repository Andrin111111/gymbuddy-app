# Privacy

## What we store
- Profile fields: name, gym, training level, goals, preferred times, contact.
- Workout data: date, duration, exercises/sets, notes, buddy id.
- No address fields, no geo coordinates, no distance data.

## Visibility
- Visibility controls if your profile appears in buddy search.
- Contact info is only shown when visibility allows it.

## Data scope
- Buddy search uses only the profile fields above.
- No geocoding provider is used.
